# Auto Jump Blast

Auto Jump Blast is a browser-based arcade platformer inspired by
endless runner mechanics. The player must time jumps precisely
to avoid obstacles while the game speed increases over time.

## Features
- Auto-run gameplay
- Single & double jump mechanics
- Increasing difficulty
- Score tracking

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## How to Run Locally
npm install
npm run dev
